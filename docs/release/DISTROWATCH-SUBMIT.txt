Description (English):
Bangla OS is a Bengali-first Linux distribution based on Debian 12 Bookworm.
It targets Bangladeshi users and Windows migrants with XFCE, OpenBangla Keyboard,
Firefox ESR, Wine, and a Calamares graphical installer. Version 1.0.1 adds
custom Padma branding: wallpaper, Plymouth/GRUB boot themes, and Calamares
installer artwork. Live session login: live / evolution. ISO ~2.3 GB.
