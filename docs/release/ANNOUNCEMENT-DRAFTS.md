**Title:** [Release] Bangla OS 1.0 (Padma) — Bengali-first Debian 12 + XFCE live ISO

**Body:**
We built **Bangla OS**, a Debian 12 Bookworm live/install ISO aimed at Bengali users and Windows migrants.

- XFCE + LightDM, Windows-style panel
- OpenBangla Keyboard (ibus)
- Firefox ESR, Wine, Calamares
- ~2.3 GB ISO (slim rebuild)
