## Reddit r/linux (English)

**Title:** [Release] Bangla OS 1.0.1 (Padma) — Bengali-first Debian 12 + XFCE, now with custom branding

**Body:**

We shipped **Bangla OS 1.0.1**, a branding refresh of our Bengali-first Debian 12 Bookworm live/install ISO.

**What's new in 1.0.1**
- Padma river wallpaper (LightDM + desktop)
- Custom logo icons and Calamares installer theme
- Slim ~2.3 GB ISO (down from ~3.5 GB in early builds)

**Stack:** XFCE + LightDM, OpenBangla (ibus), Firefox ESR, Wine, Calamares

**Download:** https://github.com/aicmsbd/bangla-os/releases/tag/v1.0.1  
BitTorrent + SHA256 on the release. Full ISO is **not** on GitHub (2 GB file limit) — use torrent or build from source.

**Live session:** `live` / `evolution`  
**Boot test:** QEMU recommended (VirtualBox may hang at Plymouth)

Feedback and issues: https://github.com/aicmsbd/bangla-os/issues
